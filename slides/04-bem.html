Let's focus on BEM

Block Element Modifier

The idea is to have naming conventions ensuring classes are unique but still easy to classify and name.

.app__todo-list--selected {
  background: #ccc;
  color: #333;
}

Those are the possible base syntax for BEM classes. Avoid using other structures if possible (except functional classes) 

.block {}

.block__element {}

.block--modifier {}

.block__element--modifer {}
My experience is that the strict model works well at scale, although with helper classes.
