CSS variables == CSS custom properties

CSS Variables are subject to the cascade, and inherit their value from their parent.

Basic usage

Declaring a variable: 
element {
  --main-bg-color: brown;
}
Using a variable: 
element {
  background-color: var(--main-bg-color);
}
Variables allow easy theming, without the need to setup and run the compile chain.
Components can easily use them as "API". 
.two {
    color: var(--my-var, red); /* Red if --my-var is not defined */
  }

Easy to interact with in JavaScript

// get variable from inline style
element.style.getPropertyValue("--my-var");

// get variable from wherever
getComputedStyle(element).getPropertyValue("--my-var");

// set variable on inline style
element.style.setProperty("--my-var", jsVar + 4);
