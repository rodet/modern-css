7. CSS Modules üè≠

"A CSS Module is a CSS file in which all class names and animation names are scoped locally by default."

"CSS Modules compile to a low-level interchange format called ICSS or Interoperable CSS, but are written like normal CSS files."

Integrations exist for:

  • PostCSS
  • Webpack
  • Vuejs

Usage example in Vue

<template>
  <header :class="$style.title">
    Title
  </header>
</template>

<style module>
.title {
  font-size: 36px;
}
</style>

The style above can only be referenced from this component

You can compose from other classes:

.otherClassName {
  composes: className from "./style.css";
}

The high granularity can be useful for refactoring, in legacy codebases...
